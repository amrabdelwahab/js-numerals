
DESCRIPTION
-----
When user gives an arabic number, the system shows the english phrase of the number.

For example:
<pre>
7    = seven
42   = forty-two
2001 = two thousand and one
1999 = nineteen hundred and ninety-nine
</pre>


INSTALL
=======

`npm install`
`bower install`

`node index` - to run
`npm test` - to test
`gulp sass` - to compile css
